# Matrix Tutoring Academy

High-fidelity static website for the AI-Native Computer Science Foundations program.

## Live Site

This repository is configured for GitHub Pages deployment through GitHub Actions.

Expected Pages URL:

`https://mitraboga.github.io/MatrixTutoringAcademy/`

## Pages Setup

The deployment workflow lives at:

`.github/workflows/deploy-pages.yml`

It deploys the repository root on every push to `main`.

If GitHub Pages is not already enabled for this repository, open:

`Settings -> Pages -> Build and deployment`

Then set the source to **GitHub Actions**.

## Site Entry Points

- `index.html` - main landing page
- `calendar.html` - cohort consultation booking page
- `syllabus.html` - syllabus stub
- `pricing.html` - pricing stub
- `testimonials.html` - testimonials stub
- `contact.html` - contact stub
